# Data Collection Using Selenium

## Project Description
This project demonstrates **data collection from websites using Selenium** in Python. The main objective is to extract structured information about products and news, handle dynamic content, navigate pages, scroll through content, and download associated images.

## Tasks Performed
- Configured Chrome WebDriver with Selenium for automated browsing.
- Extracted product names, prices, descriptions, and image URLs from multiple websites.
- Implemented **automatic scrolling** to load dynamic content on pages with infinite scroll.
- Parsed HTML using **BeautifulSoup** to collect relevant data.
- Stored the scraped data in **CSV and Excel** files.
- Downloaded product images and organized them in a dedicated folder.
- Visualized the collected images and data using **Matplotlib** and **PIL**.

## Skills Highlighted
- **Python programming** for web automation and scraping.
- **Selenium** for browser automation and handling dynamic content.
- HTML parsing with **BeautifulSoup**.
- Handling multi-page navigation and dynamic scrolling.
- Data cleaning, organization, and storage using **Pandas**.
- Image downloading and visualization using **Requests, PIL, and Matplotlib**.
